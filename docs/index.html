Features

Everything you need to
code with AI

A complete environment for AI-assisted development — from conversation to deployment.

💬

Codebase-aware Chat

Every message automatically includes your most relevant source files as context. Supports Chat (continuous history) and Ask (one-shot) modes with real-time streaming.

⚡

KODA — CEO Agent

Orchestrates specialized Code, Review and Git agents in sequence. Generates a plan with complexity rating, executes step-by-step and injects each result into the next.

🎙

Voice Control

Say "KODA" to wake the agent, describe your task, and confirm with "yes". Whisper transcription + TTS feedback for a fully hands-free workflow.

📝

Built-in Editor

CodeMirror 6 with Dracula theme, syntax highlighting for 15+ languages, Git colorization per file, and one-click Explain / Review / Edit-with-AI actions.

🌐

Live Browser Preview

Run npm run dev inside the app and see the preview in an embedded iframe. Port is auto-detected from process output.

🔀

Git Panel

Stage, commit (with AI-generated messages), pull, push, switch branches and inspect diffs — all without leaving the app. Full colored diff viewer included.

🤖

Custom Agents

Create agents with custom system prompts, assigned models and skill sets. Built-in presets: Code Reviewer, Security Analyst, Test Writer, Refactor Pro and more.

🏢

Workspaces

Each workspace has its own project root, LLM configuration and chat history. Switch between multiple projects with isolated settings per team or client.

🔧

Project Context

Drop a .aicontext file in your project root to inject conventions, key files and stack info into every LLM request automatically.

CLI

From the terminal,
right now.

Every feature available from the command line. Works without the desktop app.

koda ask "how does auth work?"

Ask a question with full codebase context injected automatically.

koda edit src/api.ts -i "add input validation"

Edit a file with a natural-language instruction.

koda review src/auth.ts

Get a detailed code review with issues and suggestions.

koda commit

Generate an AI commit message based on the current diff and commit.

koda task "add JWT auth and push"

Multi-step task executed by the CEO agent pipeline.

koda chat

Interactive chat session with history, file attachment and model switching.

koda open ~/projects/api

Open the desktop app for any directory from the terminal.

koda workspace new

Create a new isolated workspace with its own LLM config.
